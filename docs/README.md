# MandorHub

**Sistem Kontrol Proyek Harian untuk Mandor & Kontraktor**  
*Professional. Fast. Beautiful.*

![MandorHub](https://mandorhub.vercel.app/og-image.png) <!-- nanti kita update -->

## ✨ Visi

MandorHub adalah platform kontrol proyek harian yang dirancang untuk mandor dan kontraktor Indonesia.  
UI/UX terinspirasi dari **iOS + Hacker Terminal** — clean, powerful, dan terasa premium. Dibangun dengan prinsip **"less noise, maximum clarity"**.

## 🚀 Tech Stack

- **Frontend**: Next.js 15 (App Router) + TypeScript + Tailwind CSS + Framer Motion
- **Backend**: Python (FastAPI / Flask) + PostgreSQL
- **Database**: PostgreSQL (Supabase / Render)
- **Deployment**: Vercel (Frontend) + Render / Railway (Backend)
- **Design System**: GlassHacker OS (glassmorphism + iOS polish + hacker precision)
